Example


The input can be either a file or single peptide or both

A peptide is a sequence of 9 letters from the alphabet with the exceptions of the letters B, J, O, U, X, Z.
For instance, ATGPLTTLW is a peptide compatible with our model.

A valid input file should be in this csv format with one column and no header.
The column should contain peptides as describe above.
