Help


The purpose of this website is to compute the binding properties of a peptide using a Feed Forward Neural Network model.
The "concentration refers to the IC50 value, the concentration of an inhibitor where the binding is reduced by half.
The "probability" refers to the probability of presentation of the peptide by the MHC system at the surface of the cell.

Input:
The input can be a single peptide or a csv file. Its details appear in the "Example" section.

Output:
The model returns the "concentration" and the "probability" for each peptide. If the input is a csv file, a csv file with the results can be downloaded.
